# 英语成神之路 - English Mastery

一个简约现代的英语单词学习Web应用，帮助你通过闪卡复习和系统管理来掌握英语单词。

## 功能特性

### 📝 核心功能
- **单词管理**：添加、编辑、删除英语单词和短语
- **闪卡复习**：正反面翻转卡片，测试记忆效果
- **智能复习**：基于记忆曲线安排复习计划
- **标签分类**：为单词添加标签，方便分类管理
- **搜索筛选**：快速查找需要的单词

### 📊 学习统计
- **进度追踪**：记录学习进度和掌握程度
- **学习趋势**：图表展示学习历史
- **成就系统**：解锁学习成就，保持动力
- **连续学习**：记录连续学习天数

### 🎨 用户体验
- **现代简约设计**：美观的UI界面
- **响应式布局**：支持手机、平板和电脑
- **离线使用**：数据本地存储，无需网络
- **PWA支持**：可安装到桌面，像原生应用一样使用
- **键盘快捷键**：提高操作效率

### 🔒 数据安全
- **本地存储**：所有数据保存在浏览器本地
- **隐私保护**：无需注册登录，数据完全私有
- **导入导出**：支持数据备份和恢复
- **自动保存**：定期自动保存，防止数据丢失

## 技术栈

- **前端**：纯HTML/CSS/JavaScript
- **存储**：LocalStorage + IndexedDB
- **图表**：Chart.js
- **图标**：Font Awesome 6
- **字体**：Inter (Google Fonts)
- **PWA**：Service Worker + Web App Manifest

## 快速开始

### 在线使用
访问部署地址：https://[你的用户名].github.io/english-mastery/

### 本地运行
1. 克隆仓库：
   ```bash
   git clone https://github.com/[你的用户名]/english-mastery.git
   cd english-mastery
   ```

2. 安装依赖（可选）：
   ```bash
   npm install
   ```

3. 启动开发服务器：
   ```bash
   npm run dev
   ```

4. 在浏览器中打开：http://localhost:3000

## 使用指南

### 1. 添加单词
1. 点击"添加单词"标签页
2. 输入英文单词和中文释义
3. （可选）添加例句和标签
4. 点击"保存单词"

### 2. 复习单词
1. 点击"闪卡复习"标签页
2. 选择复习范围（所有单词/需要复习/按标签）
3. 点击"开始复习"
4. 点击卡片翻转查看释义
5. 根据记忆难度选择"困难"或"简单"

### 3. 管理单词
1. 点击"单词列表"标签页
2. 使用搜索框快速查找
3. 按标签筛选单词
4. 点击编辑或删除按钮管理单词

### 4. 查看统计
1. 点击"学习统计"标签页
2. 查看学习进度和成就
3. 分析学习趋势图表
4. 查看标签分布

## 键盘快捷键

- `Ctrl/Cmd + N`：添加新单词
- `Ctrl/Cmd + F`：搜索单词
- `Ctrl/Cmd + R`：开始复习
- `Ctrl/Cmd + S`：保存数据
- `空格/回车`：翻转卡片
- `左箭头`：上一张卡片
- `右箭头`：下一张卡片
- `1/H`：标记为困难
- `2/E`：标记为简单

## 数据管理

### 导出数据
1. 点击"单词列表"页面的"导出数据"按钮
2. 数据将以JSON格式下载

### 导入数据
1. 点击"单词列表"页面的"导入数据"按钮
2. 选择之前导出的JSON文件
3. 确认导入

### 清空数据
1. 点击"单词列表"页面的"清空所有"按钮
2. 确认操作（此操作不可撤销）

## 开发指南

### 项目结构
```
english-mastery/
├── src/
│   ├── index.html          # 主页面
│   ├── css/
```

3. 在 `app.js` 中初始化模块
